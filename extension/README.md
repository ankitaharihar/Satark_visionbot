# Satark Browser Extension (Chrome)

## Load extension

1. Open Chrome and go to chrome://extensions
2. Enable Developer mode
3. Click Load unpacked
4. Select the extension folder

## Usage

1. Ensure Satark web API is running on http://127.0.0.1:8000
2. Open any page
3. Click extension icon
4. Analyze URL

## Notes

- Extension reads current tab URL and sends it to local API.
- Keep API running before using extension.
